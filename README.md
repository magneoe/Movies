# Movies


## Overview & Installation
The project is written with Java jdk8, using the Spring Boot project.
Frontend is setup using JSP.

1) Make sure Maven dependencies are being downloaded and the project is built locally.
2) From a IDE (e.g. Eclipse), run as a regular Java application.

Note: The project comes with an embedded Tomcat server running at port 8080, and a H2 database located at project-root/movies.mv.db.

To manually inspect the DB, goto localhost:8080/h2
